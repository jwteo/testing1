K-Means Clustering

K-Means divides your dataset into k number of clusters using an iterative optimisation strategy. Each cluster is represented by a centre. A point belongs to to a cluster whose centre is closest to it.

Meauring Model performance

We mainly want to look at how well the clusters are formed in clustering. Ideally, we want the clusters to be well separated, large but dense. Yu can get the largest cluster by setting k (the number of clusters) as small as one. You can get the densest clusters by using a large k, creating many dense micro-clusters. We lose any insightful information we might want to infer from the cluster in both cases.
